- HTML
- CSS

## 🦮 Instruções Gerais para utilização da Interface Web


- Com as setas de direção (➡️, ⬅️, ⬆️, ⬇️) movimente a cobrinha verde para o alimento vermelho. Caso a cobrinha toque no próprio corpo o jogo acaba!
